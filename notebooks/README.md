Task 1 — Data Collection and Preprocessing

Objective

The objective of Task 1 was to scrape customer reviews from the Google Play Store and preprocess them into a clean, analysis-ready dataset.

Activities Performed

- Scraped reviews from:
  - CBE Mobile Banking
  - BOA Mobile Banking
  - Dashen Super App
- Collected:
  - review text
  - ratings
  - review dates
  - bank names
  - review source
- Removed duplicates
- Handled missing values
- Standardized dates
- Exported cleaned CSV dataset

Tools Used

- google-play-scraper
- pandas
- matplotlib
- seaborn

Outputs

- Cleaned review dataset
- Review distribution visualizations
- Rating analysis
- Review length analysis
  Task 2 — Sentiment and Thematic Analysis

Objective

The objective of Task 2 was to analyze customer sentiment and identify recurring themes within mobile banking app reviews.

Activities Performed

- Applied sentiment analysis using VADER
- Generated:
  - sentiment scores
  - sentiment labels
- Performed keyword extraction using TF-IDF
- Identified recurring themes:
  - Account Access Issues
  - Transaction Performance
  - UI & Design
  - OTP Issues
- Visualized:
  - sentiment distribution
  - keyword frequency
  - theme frequency

Tools Used

- NLTK VADER
- Scikit-learn TF-IDF
- Matplotlib
- Seaborn
- WordCloud

Outputs

- Sentiment-labeled review dataset
- Keyword analysis
- Theme categorization
- Business insight visualizations
